# 🎨 CSS Псевдоэлементы

> Конспект по псевдоэлементам CSS.

---

## 📌 Что такое псевдоэлемент?

**Псевдоэлемент** позволяет создать новый элемент с помощью CSS, **не добавляя его в HTML**.

Пишется через `::` (два двоеточия).

```css
.card::before {
}

.card::after {
}
```

---

## 🔄 Чем псевдоэлемент отличается от псевдокласса?

### Псевдокласс

Изменяет **состояние** элемента.

```css
button:hover {
  background: blue;
}
```

При наведении кнопка становится синей.

### Псевдоэлемент

Создаёт **новый виртуальный элемент**.

```css
.card::before {
  content: "NEW";
}
```

Хотя в HTML слова `NEW` нет.

---

## ⬅️ `::before`

Добавляет элемент **перед** содержимым.

**HTML**

```html
<p class="price">999$</p>
```

**CSS**

```css
.price::before {
  content: "💰";
}
```

**Результат:**

```
💰999$
```

---

## ➡️ `::after`

Добавляет элемент **после** содержимого.

**HTML**

```html
<p class="price">999$</p>
```

**CSS**

```css
.price::after {
  content: "🔥";
}
```

**Результат:**

```
999$🔥
```

---

## 🧩 `content`

**Главное свойство** псевдоэлементов.

Без него `::before` и `::after` работать не будут.

Можно написать:

```css
content: "NEW";
```

или

```css
content: "⭐";
```

или

```css
content: "";
```

> Даже пустая строка считается содержимым.

---

## 🔀 Использование сразу двух псевдоэлементов

```css
.price::before {
  content: "💰";
}

.price::after {
  content: "🔥";
}
```

Получится:

```
💰999$🔥
```

---

## 📍 `position: relative`

Используется у **родителя**.

```css
.card {
  position: relative;
}
```

Он становится **точкой отсчёта** для элементов с `position: absolute`.

---

## 📌 `position: absolute`

Используется у **дочернего элемента** или **псевдоэлемента**.

```css
.card::before {
  content: "NEW";
  position: absolute;
  top: 10px;
  left: 10px;
}
```

Теперь можно разместить элемент в любом месте карточки.

---

## 📏 `top`, `bottom`, `left`, `right`

Используются вместе с `position: absolute`.

```css
top: 10px;
left: 10px;
```

или

```css
top: 0;
right: 0;
```

Они задают **расстояние от края** родительского элемента.

---

## 📐 `display: block`

Часто используется вместе с `::after`.

```css
h2::after {
  content: "";
  display: block;
  width: 100px;
  height: 3px;
  background: red;
}
```

Получится декоративная линия под заголовком.

---

## 📍 Где используются псевдоэлементы?

Очень часто, например:

- значок `NEW`
- значок `SALE`
- декоративные линии
- подчёркивания
- иконки
- стрелки
- звёздочки
- оформление карточек товаров

---

## 🐞 Частые ошибки

### ❌ Забыть написать `content: ""`

Без `content` псевдоэлемент не появится.

---

### ❌ Написать `position: absolute`, но забыть `position: relative` у родителя

Тогда элемент может расположиться относительно всей страницы.

---

### ❌ Использовать `p::before`, когда нужен только один элемент

Лучше использовать класс:

```css
.price::before
```
