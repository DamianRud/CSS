# 📐 Grid: выравнивание элементов

Два свойства, две оси. Каждое работает **только внутри `display: grid`**.

---

## `justify-items` — по горизонтали

Управляет положением элемента вдоль строки (влево/вправо).

```css
justify-items: start;
```
⬅️ Слева

```css
justify-items: center;
```
🎯 По центру

```css
justify-items: end;
```
➡️ Справа

---

## `align-items` — по вертикали

Управляет положением элемента поперёк строки (вверх/вниз).

```css
align-items: start;
```
⬆️ Сверху

```css
align-items: center;
```
🎯 По центру

```css
align-items: end;
```
⬇️ Снизу

---

## 🧠 Шпаргалка

| Свойство | Ось | `start` | `center` | `end` |
|---|---|---|---|---|
| `justify-items` | ⟷ горизонталь | слева | по центру | справа |
| `align-items` | ↕ вертикаль | сверху | по центру | снизу |

> 💡 Если нужно задать оба сразу — есть шорткат `place-items: <align> <justify>;`
