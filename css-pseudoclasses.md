# 🎨 CSS Псевдоклассы

## Что такое псевдокласс?

**Псевдокласс** — это специальное состояние элемента.

Он позволяет изменить стиль элемента без JavaScript.

Пишется через `:` (одно двоеточие).

**Пример:**

```css
button:hover {
    background: blue;
}
```

---

## 📌 `:hover`

Срабатывает, когда курсор мыши находится над элементом.

```css
.button:hover {
    background: blue;
}
```

Используется для:

- кнопок
- ссылок
- карточек товаров
- изображений

---

## 📌 `:active`

Срабатывает в момент нажатия на элемент.

```css
.button:active {
    background: red;
    transform: scale(0.95);
}
```

Используется для создания эффекта нажатия.

---

## 📌 `:focus`

Срабатывает, когда элемент получает фокус.

Чаще всего используется у:

- `input`
- `textarea`
- `button`

**Пример:**

```css
input:focus {
    border: 2px solid blue;
    outline: none;
}
```

Можно добавить красивую подсветку:

```css
input:focus {
    border: 2px solid dodgerblue;
    box-shadow: 0 0 10px lightblue;
}
```

---

## 📌 `:first-child`

Выбирает первого ребёнка внутри родителя.

**HTML:**

```html
<div class="card">1</div>
<div class="card">2</div>
<div class="card">3</div>
```

**CSS:**

```css
.card:first-child {
    background: red;
}
```

**Результат:**

```
🟥 1
⬜ 2
⬜ 3
```

---

## 📌 `:last-child`

Выбирает последнего ребёнка.

```css
.card:last-child {
    background: blue;
}
```

**Результат:**

```
⬜ 1
⬜ 2
🟦 3
```

---

## 📌 `:nth-child()`

Позволяет выбрать элемент по номеру.

Например:

```css
.card:nth-child(3) {
    background: yellow;
}
```

**Результат:**

```
⬜ 1
⬜ 2
🟨 3
⬜ 4
```

### Чётные элементы

```css
.card:nth-child(even)
```

Выбирает:

```
2
4
6
8
...
```

### Нечётные элементы

```css
.card:nth-child(odd)
```

Выбирает:

```
1
3
5
7
...
```

---

## Где используются псевдоклассы?

- Кнопки
- Формы
- Меню
- Таблицы
- Карточки товаров
- Списки
- Навигация

---

## ⚠️ Частые ошибки

### ❌ Путать `:hover` и `:active`

- `:hover` — курсор находится над элементом.
- `:active` — кнопка нажата.

### ❌ Забывать `outline: none`

Иногда браузер рисует стандартную рамку вокруг `input`.

Если нужен свой стиль:

```css
input:focus {
    outline: none;
}
```

### ❌ Использовать сразу несколько псевдоклассов при проверке

Например:

```css
.card:first-child {
    background: red;
}

.card:nth-child(odd) {
    background: blue;
}
```

Первая карточка подходит под оба правила. Если селекторы имеют одинаковую специфичность, применится правило, которое написано ниже.
