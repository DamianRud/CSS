# 📐 CSS Grid — конспект

## 1. `grid-column`

**Что делает?** Управляет тем, сколько колонок занимает элемент.

```css
.big {
  grid-column: span 2;
}
```

`span 2` → элемент займёт **2 колонки**.

Было:
```text
[1][2][3]
[4][5][6]
```

Стало (`.big` — это ячейка 1):
```text
[   1   ][2]
[  3  ][4][5]
[   6   ]
```

> `grid-column` делает элемент **шире**.

---

## 2. `grid-row`

**Что делает?** Управляет тем, сколько строк занимает элемент.

```css
.big {
  grid-row: span 2;
}
```

`span 2` → элемент займёт **2 строки**.

Было:
```text
[1][2][3]
[4][5][6]
```

Стало (`.big` — это ячейка 1):
```text
[ 1 ][2][3]
[ 1 ][4][5]
[   ][6]
```

> `grid-row` делает элемент **выше**.

---

## 3. `grid-template-rows`

**Что делает?** Создаёт строки и задаёт их высоту.

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: 100px 200px;
}
```

| Строка | Высота |
|---|---|
| 1-я | 100px |
| 2-я | 200px |

Если написать:
```css
grid-template-rows: 150px 150px 150px;
```
получится **3 строки** по **150px** каждая.

---

## 🧠 Что важно запомнить

| Свойство | Эффект |
|---|---|
| `grid-column: span 2;` | делает элемент **шире** (2 колонки) |
| `grid-row: span 2;` | делает элемент **выше** (2 строки) |
| `grid-template-rows: 100px 200px;` | задаёт строки и их высоту |
