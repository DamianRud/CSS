# 📱 CSS `@media` — Адаптивная вёрстка

> Конспект по адаптивной вёрстке с помощью медиа-запросов CSS.

---

## 📌 Что такое `@media`?

`@media` — это правило CSS, которое позволяет **менять стили в зависимости от размера экрана устройства**.

Используется для создания адаптивной вёрстки, чтобы сайт хорошо выглядел на компьютере, планшете и телефоне.

---

## ✍️ Синтаксис

```css
@media (max-width: 768px) {
  /* CSS для экранов до 768px */
}
```

| Часть | Значение |
|---|---|
| `@media` | правило для адаптивных стилей |
| `max-width` | максимальная ширина экрана |
| `768px` | если ширина экрана 768px или меньше — применятся эти стили |

---

## 🖼️ Пример

### Компьютер

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
}
```

Получится:

```
📦 📦 📦
📦 📦 📦
```

### Телефон

```css
@media (max-width: 768px) {
  .container {
    grid-template-columns: 1fr;
  }
}
```

Получится:

```
📦
📦
📦
```

---

## 🔢 Несколько `@media`

Можно задавать стили сразу для нескольких размеров экранов:

```css
.container {
  grid-template-columns: repeat(4, 1fr);
}

@media (max-width: 1024px) {
  .container {
    grid-template-columns: repeat(2, 1fr);
  }
}

@media (max-width: 768px) {
  .container {
    grid-template-columns: 1fr;
  }
}
```

**Результат:**

| Устройство | Колонки |
|---|---|
| 💻 Компьютер | 4 |
| 📱 Планшет | 2 |
| 📱 Телефон | 1 |

---

## 🎛️ Что можно менять через `@media`

Практически любые CSS-свойства, например:

- `padding`
- `margin`
- `width`
- `height`
- `font-size`
- `display`
- `grid-template-columns`
- `grid-template-rows`
- `grid-template-areas`
- `gap`

---

## 👻 `display: none`

Используется, чтобы **полностью скрыть элемент**.

```css
@media (max-width: 768px) {
  .aside {
    display: none;
  }
}
```

До 768px блок исчезнет.

---

## 📐 `auto`

Используется, чтобы размер **автоматически подстраивался** под содержимое.

```css
grid-template-rows: 100px auto 100px;
```

Если контента станет больше — строка автоматически увеличится.

---

## 📋 Порядок написания

1. Сначала стили для **компьютера**
2. Потом адаптация для **планшета**
3. Потом адаптация для **телефона**

```css
.container {
  grid-template-columns: repeat(4, 1fr);
}

@media (max-width: 1024px) {
  .container {
    grid-template-columns: repeat(2, 1fr);
  }
}

@media (max-width: 768px) {
  .container {
    grid-template-columns: 1fr;
  }
}
```

---

## 📏 Полезные размеры экранов

| Устройство | Размер |
|---|---|
| 💻 Компьютер | больше 1024px |
| 📱 Планшет | до 1024px |
| 📱 Телефон | до 768px |

> ⚠️ Это самые распространённые значения, но в разных проектах они могут отличаться.

---

## 🐞 Частые ошибки

### ❌ Неправильно

```css
grid-template-columns: 2fr;
```

Это **одна** колонка.

### ✅ Правильно

```css
grid-template-columns: repeat(2, 1fr);
```

Это **две** колонки.

---

### ❌ Забыли изменить сетку внутри `.main`

```css
@media (max-width: 768px) {
  .container {
    grid-template-columns: 1fr;
  }
}
```

Если карточки находятся внутри `.main`, нужно менять и её сетку тоже.

---

## 💡 Главное правило

> На компьютере пишем **основные стили**.
> В `@media` изменяем **только то**, что должно измениться на другом размере экрана.
> Не нужно заново переписывать все свойства.
