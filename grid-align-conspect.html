Выравнивание в Grid

Два свойства, одна ось у каждого. Нажимай на кнопки — коробка двигается по своей чертёжной сетке в реальном времени.

justify-items

Управляет положением элемента по горизонтали (вдоль строки).

⬅️

justify-items: start;

align-items

Управляет положением элемента по вертикали (поперёк строки).

⬆️

align-items: start;

⟷ justify-items

  • start — слева
  • center — по центру
  • end — справа

↕ align-items

  • start — сверху
  • center — по центру
  • end — снизу
display: grid — работает только внутри grid-контейнера
