# CSS Grid — конспект

## 1. Что такое Grid

`display: grid` / `display: inline-grid` — включает grid-контекст для элемента. Все прямые дети становятся grid-элементами (grid items), а сам элемент — grid-контейнером (grid container).

---

## 2. Свойства контейнера (grid container)

### Задание сетки

| Свойство | Описание |
|---|---|
| `grid-template-columns` | размеры и количество колонок |
| `grid-template-rows` | размеры и количество строк |
| `grid-template-areas` | именованные области сетки (раскладка "картой") |
| `grid-template` | сокращение: rows + columns + areas |
| `grid` | супер-сокращение: template + auto-flow + auto-rows/columns |

### Промежутки

| Свойство | Описание |
|---|---|
| `gap` (`grid-gap`) | сокращение row-gap + column-gap |
| `row-gap` (`grid-row-gap`) | промежуток между строками |
| `column-gap` (`grid-column-gap`) | промежуток между колонками |

### Неявная сетка (когда элементов больше, чем задано ячеек)

| Свойство | Описание |
|---|---|
| `grid-auto-columns` | размер автоматически создаваемых колонок |
| `grid-auto-rows` | размер автоматически создаваемых строк |
| `grid-auto-flow` | направление заполнения: `row` \| `column` \| `dense` |

### Выравнивание содержимого ячеек (внутри контейнера)

| Свойство | Описание |
|---|---|
| `justify-items` | выравнивание элементов по горизонтали внутри ячеек |
| `align-items` | выравнивание элементов по вертикали внутри ячеек |
| `place-items` | сокращение align-items + justify-items |

### Выравнивание всей сетки (если она меньше контейнера)

| Свойство | Описание |
|---|---|
| `justify-content` | выравнивание сетки по горизонтали |
| `align-content` | выравнивание сетки по вертикали |
| `place-content` | сокращение align-content + justify-content |

> ⚠️ Важный момент: `justify-items`, `align-items`, `justify-content`, `align-content` и их `self`-версии — это НЕ свойства flexbox. Они из отдельного стандарта **CSS Box Alignment**, который использует и Flexbox, и Grid. Разница в том, что во Flexbox `justify-items`/`justify-self` **не работают** (нет фиксированных ячеек), а в Grid — работают.

---

## 3. Свойства элементов (grid items)

### Позиционирование по линиям сетки

| Свойство | Описание |
|---|---|
| `grid-column-start` | с какой вертикальной линии начинается |
| `grid-column-end` | на какой вертикальной линии заканчивается |
| `grid-row-start` | с какой горизонтальной линии начинается |
| `grid-row-end` | на какой горизонтальной линии заканчивается |
| `grid-column` | сокращение start/end через `/` (например `1 / 3`) |
| `grid-row` | сокращение start/end через `/` |
| `grid-area` | сокращение всех 4 линий, либо имя из `grid-template-areas` |

### Индивидуальное выравнивание элемента

| Свойство | Описание |
|---|---|
| `justify-self` | переопределяет `justify-items` для одного элемента |
| `align-self` | переопределяет `align-items` для одного элемента |
| `place-self` | сокращение align-self + justify-self |

---

## 4. Полезные функции и значения

| Функция/значение | Описание |
|---|---|
| `repeat(n, размер)` | повтор колонок/строк, например `repeat(3, 1fr)` |
| `minmax(min, max)` | задаёт диапазон допустимого размера |
| `fr` | единица "доля свободного места" |
| `auto-fill` / `auto-fit` | вместе с `repeat()` для адаптивных сеток без медиа-запросов |
| `subgrid` | элемент-сетка наследует линии родительской сетки |

---

## 5. Минимальный рабочий пример

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 16px;
}

.item {
  grid-column: 1 / 3; /* занимает 2 колонки */
}
```

---

## 6. Шпаргалка "что чаще всего реально нужно"

1. `display: grid`
2. `grid-template-columns`
3. `gap`
4. `grid-column` / `grid-row`
5. `grid-template-areas` (для сложных макетов)

Остальное — для тонкой настройки под конкретные задачи.
